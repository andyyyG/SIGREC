# Sistema de Gestión Integral para el Monitoreo y Regulación del Comercio

Sistema web desarrollado para la gestión de vendedores y solicitudes, permitiendo el control y seguimiento de procesos administrativos mediante validación y manejo de datos.

## Tecnologías utilizadas
- Node.js
- Express
- MySQL
- JavaScript

## Funcionalidades principales
- Registro y verificación de usuarios
- Gestión de solicitudes (nuevo vendedor, renovación y productos)
- Validación y control de información
- Aprobación y rechazo de solicitudes
- Manejo de base de datos relacional

## Objetivo
Optimizar la gestión y seguimiento de solicitudes comerciales, asegurando la integridad y control de la información.

## Ejecución del proyecto
1. Clonar el repositorio
2. Instalar dependencias: npm install
3. Ejecutar el servidor: node app.js
